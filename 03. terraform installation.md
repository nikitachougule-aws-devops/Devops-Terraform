## Terraform Installation

Terraform installation is the process of downloading and setting up the Terraform CLI (Command Line Interface) on your computer.

After installation, you can create and manage infrastructure using Terraform commands.

---

## Prerequisites

Before installing Terraform, make sure you have:

* A computer running Windows, Linux, or macOS
* Internet connection
* Terminal or Command Prompt
* A cloud account (AWS, Azure, or GCP) for creating resources (optional for learning)

---

## Install Terraform on Windows

### Step 1: Download Terraform

Download the latest Terraform ZIP file from the official HashiCorp website.

---

### Step 2: Extract the ZIP File

Extract the downloaded ZIP file.

You will get:

```text
terraform.exe
```

---

### Step 3: Add Terraform to PATH

1. Copy `terraform.exe` to a folder (for example: `C:\Terraform`).
2. Add that folder to the Windows **Environment Variables (PATH)**.
3. Restart Command Prompt or PowerShell.

---

### Step 4: Verify Installation

Open Command Prompt and run:

```bash
terraform version
```

Example output:

```text
Terraform v1.x.x
```

---

## Install Terraform on Linux

Download Terraform:

```bash
wget https://releases.hashicorp.com/terraform/<version>/terraform_<version>_linux_amd64.zip
```

Extract the file:

```bash
unzip terraform_<version>_linux_amd64.zip
```

Move Terraform to the system path:

```bash
sudo mv terraform /usr/local/bin/
```

Verify installation:

```bash
terraform version
```

---

## Install Terraform on macOS

Using Homebrew:

```bash
brew tap hashicorp/tap
brew install hashicorp/tap/terraform
```

Verify installation:

```bash
terraform version
```

## Terraform Commands

```bash
terraform version
terraform init
terraform plan
terraform apply
terraform destroy
```

---

## Upgrade Terraform

Download the latest version and replace the existing Terraform executable.

Verify the new version:

```bash
terraform version
```

---

## Uninstall Terraform

### Windows

* Delete `terraform.exe`.
* Remove its folder from the **PATH** environment variable.

### Linux

```bash
sudo rm /usr/local/bin/terraform
```

### macOS

```bash
brew uninstall terraform
```

---

## Installation Verification

Run:

```bash
terraform version
```

If Terraform is installed correctly, it displays the installed version.

You can also check the available commands:

```bash
terraform
```

This displays the list of supported Terraform commands.

---

## Key Points

* Install the Terraform CLI on your operating system.
* Verify the installation using `terraform version`.
* Add Terraform to the system **PATH** if required.
* Terraform is managed through the command line.
* After installation, the next step is to initialize a project using `terraform init`.
